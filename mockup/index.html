How to play this game?

To make a line of four disks in a row horizontally, vertically or diagonally. Game console with base, 23 red disks and 23 yellow disks, instructions. Assemble the frame and base and place the console between the players.

Can I install this game from Appstores?

Yes, this game is available in Appstores.It is easy to install. The steps are specified.

Can this game include more than two players?

This game can be played by two players minimum.If you want to choose one or more players then they play as partners.

Can I download from the internet?

This game can be downloaded from the internet.
